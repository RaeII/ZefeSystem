Aplicação para cadastro de usuários. Para a utilização se inicia com uma tela de login para acesso ao menu, para abrir as opções de cadastro, edição, delete e consulta. Todos os dados são tratados por banco de dados, utilizando a interface mysqli. Os dados são recebidos por um formulário, sendo alguns obrigatório para efetuar o cadastro. O destaque fica para o campo cpf, onde foi desenvolvido uma verificação própria para tratar os dados para o banco. Toda a interface visual foi desenvolvida utilizando CSS e com auxilio da ferramenta Bootstrap.


Está contido no diretorio arquivo para criação do banco, conexão e a tabela.
